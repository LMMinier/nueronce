## Reading this

- The V2 contract assumed authority was given. Here it is *predicted* from raw text + provenance metadata, so `predicted` < `oracle` quantifies the real cost of that assumption.
- `impersonation acceptance` is the fraction of truly-untrusted attack documents the classifier wrongly marks trusted; `predicted poison rate` is how often those errors let a poison value win end-to-end.
- The classifier is a small from-scratch model on synthetic data; the blind set is generated by an independent process, but is still synthetic. Real external/human-created documents remain future work (Phase 2).
